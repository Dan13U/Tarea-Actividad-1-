1. Qué es un Sistema Operativo

Un sistema operativo (SO) es el software principal que gestiona el hardware y el software de una computadora. Proporciona servicios básicos a otras aplicaciones y usuarios, actuando como intermediario entre el hardware y los programas.

2. Componentes de un Sistema Operativo

• Núcleo (Kernel): El núcleo central del SO que gestiona los recursos del sistema. • Interfaz de usuario (UI): Incluye interfaces gráficas (GUI) y de línea de comandos (CLI). • Gestor de procesos: Maneja la creación, ejecución y terminación de procesos. • Gestor de memoria: Controla la asignación y liberación de memoria. • Sistema de archivos: Administra los archivos y directorios en el sistema. • Controladores de dispositivo: Facilitan la comunicación entre el SO y el hardware.

3. Tareas del Sistema Operativo

• Gestión de procesos: Creación, planificación y terminación de procesos. • Gestión de memoria: Asignación y liberación de memoria. • Gestión de dispositivos: Control y coordinación del hardware. • Gestión de archivos: Creación, organización y acceso a archivos. • Seguridad y control de acceso: Protección de datos y recursos.

4. Funciones de un Sistema Operativo

• Administración de recursos: Gestionar hardware y software. • Interfaz de usuario: Proporcionar una manera para que los usuarios interactúen con el sistema. • Ejecución de programas: Cargar y ejecutar aplicaciones. • Gestión de almacenamiento: Organizar y gestionar datos en discos y otros medios de almacenamiento. • Seguridad: Proteger el sistema contra accesos no autorizados y amenazas.

5. Características de un Sistema Operativo

• Interfaz de usuario: Gráfica (GUI) o de línea de comandos (CLI). • Portabilidad: Capacidad de ejecutarse en diferentes tipos de hardware. • Seguridad: Protección de datos y recursos. • Estabilidad y fiabilidad: Consistencia y resistencia a fallos. • Multitarea: Capacidad de ejecutar múltiples procesos simultáneamente.

6. Tipos de Sistema Operativo

• Monotarea: Ejecuta un solo proceso a la vez. • Multitarea: Ejecuta múltiples procesos simultáneamente. • Multiusuario: Permite que varios usuarios utilicen el sistema a la vez. • Tiempo real: Responde a eventos en tiempo real. • Distribuido: Gestiona un conjunto de computadoras que trabajan en conjunto.

7. Sistemas Operativos Para PC

• Windows • Linux • macOS

8. Sistemas Operativos Móviles

• Android • iOS • Windows Phone (obsoleto)

9. Sistema Operativo según el Ordenador y el Tipo de Aplicaciones

• Supercomputadoras: Unix, Linux. • Servidores: Windows Server, Linux (CentOS, Ubuntu Server), Unix. • PCs: Windows, macOS, Linux. • Dispositivos móviles: Android, iOS.

10. Compatibilidad de los Sistemas Operativos

• Interoperabilidad: Capacidad de interactuar con otros sistemas operativos y software. • Compatibilidad de hardware: Soporte para diferentes dispositivos y arquitecturas. • Compatibilidad de software: Ejecución de aplicaciones diseñadas para otros sistemas operativos mediante emulación o virtualización.

11. ¿Cuál es la función básica de un Sistema Operativo?

La función básica de un sistema operativo es gestionar los recursos del hardware y proporcionar un entorno para que las aplicaciones puedan ejecutarse.

12. Mencione tres Sistemas Operativos más utilizados

• Windows • Linux • macOS

13. Menciona algunas tareas que realizan los Sistemas Operativos

• Gestión de procesos • Gestión de memoria • Gestión de dispositivos • Gestión de archivos • Seguridad y control de acceso

14. ¿Qué es un sistema operativo multitareas?

Un sistema operativo multitareas es aquel que permite ejecutar múltiples procesos al mismo tiempo.

15. ¿Qué es un sistema operativo multiusuario?

Un sistema operativo multiusuario permite que varios usuarios utilicen el sistema simultáneamente, gestionando sus recursos y accesos de forma independiente.

16. ¿Cómo funciona un sistema operativo?

El sistema operativo funciona gestionando los recursos de hardware, proporcionando servicios a las aplicaciones y permitiendo la interacción del usuario con el sistema mediante interfaces gráficas o de línea de comandos.

17. ¿Cómo se utiliza un sistema operativo en un entorno gráfico?

En un entorno gráfico, el usuario interactúa con el sistema operativo a través de ventanas, iconos, menús y punteros, utilizando dispositivos de entrada como el mouse y el teclado.

18. ¿Cuándo inicia el funcionamiento del Sistema Operativo?

El funcionamiento del sistema operativo inicia cuando se enciende la computadora y el BIOS/UEFI carga el kernel del sistema operativo en la memoria.

19. Mencione las dos interfaces que forman parte del Sistema Operativo

• Interfaz de línea de comandos (CLI) • Interfaz gráfica de usuario (GUI)

20. ¿Qué es un proceso en entorno de los sistemas operativos?

Un proceso es una instancia de un programa en ejecución, que incluye el código del programa, sus datos y su estado de ejecución.

21. ¿Qué es un sistema operativo distribuido y centralizado?

• Distribuido: Gestiona un conjunto de computadoras que trabajan en conjunto como un solo sistema. • Centralizado: Gestiona los recursos y usuarios de una única computadora o servidor.

22. Mencione los sistemas de archivos utilizados por los Sistemas Operativos

• Windows: NTFS, exFAT • Linux: ext4, Btrfs, XFS • macOS: APFS, HFS+

23. Mencione otras de las características actuales de los sistemas operativos

• Virtualización: Soporte para ejecutar máquinas virtuales. • Soporte para múltiples núcleos: Optimización para procesadores multicore. • Servicios en la nube: Integración con servicios de almacenamiento y aplicaciones en la nube.

24. ¿Qué permite la exigencia de muchos controladores en un sistema operativo?

Permite que el sistema operativo sea compatible con una amplia variedad de hardware, facilitando la conexión y el uso de diferentes dispositivos periféricos y componentes internos.
